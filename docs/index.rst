====================================================
coPTR: *co*\ mpute *P*\ eak-to-*T*\ rough *R*\ atios
====================================================
coPTR is a tool for computing peak-to-trough ratios from metagenomic
sequencing data. It supports estimation from both complete reference
genomes and draft assemblies.
